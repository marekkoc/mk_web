Podsumowanie - 4 sposoby

1. Linki
href="?kategoria=wartość"
2. Przyciski
button name="kategoria" value="wartość"
3. Dropdown
select name="kategoria" + onchange
4. Input tekstowy
input name="kategoria" + submit

Wszystkie sposoby robią to samo: wypełniają $_GET['kategoria'] wartością wybraną przez użytkownika!
